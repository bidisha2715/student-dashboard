# Student Dashboard

A Python-based dashboard for visualizing student performance data using graphs and charts.

## 🚀 Features
- Reads student data from CSV files
- Calculates average marks
- Identifies topper
- Bar graph visualization
- Pie chart for grade distribution

## 🛠️ Tech Stack
- Python
- Matplotlib
- CSV handling

## 💡 Description
This project focuses on data visualization and analysis using Python, providing graphical insights into student performance.
